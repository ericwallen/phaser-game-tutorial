Phaser Game tutorial

Learn how to raplidy build HTML5 games from scratch with popular game engine Phaser.

Author: Eric Wallen

Hello! Welcome to this tutorial and intro to Phaser. With this powerful library you can build HTML5 JavaScript based 2d games for the browser very quickly. Playing games is a fun activity if there ever was one. I think we can all agree to that? Creating games, believe it or not, can be even more enjoyable! You really get to own it and that is so â€¦. satisfying. Are you ready to do this!? Ask your self a few questions to determine if we have a good fit. First, do you know a little JavaScript or have coded in some language even just HTML or CSS? Have you touched the command line/prompt once or more and understand how to launch it? Great! Then you will be just fine. If you have some illustrator or design knowledge that can also be helpful but I will supply you all my images so that you can worry about skinning your game the way you want to later. Sound good? In essence this tutorial is meant to be from step one so no previous knowledge about phaser is required. Ok, lets go!
